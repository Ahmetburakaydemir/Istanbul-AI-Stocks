# Istanbul AI Stocks

Minimalist, Apple esintili bir tek sayfa uygulama. Borsa İstanbul hisselerini yapay zeka destekli kısa özetlerle, sade grafikler ve seçilebilir zaman aralıklarıyla gösterir. Veriler demo amaçlıdır ve yerel olarak çalışır.

## Özellikler
- Hızlı hisse ve zaman aralığı seçimi (BIMAS, THYAO, ASELS, GARAN örnekleri)
- Apple tarzı cam efektli kartlar ve yumuşak renk paleti
- Canvas tabanlı mini grafik (sparkline) ve AI yorum kartı
- Makro vurgularla kompakt okuma deneyimi

## Çalıştırma
Herhangi bir paket kurulumu gerekmez. Depodaki `index.html` dosyasını doğrudan tarayıcıda açın veya basit bir sunucu başlatın:

```bash
python -m http.server 8000
# ardından tarayıcıda http://localhost:8000 adresini açın
```
